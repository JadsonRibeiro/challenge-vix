## Tasks Register

Aplicação desenvolvida para o desafio para devs Full Stack da [Vix](https://github.com/vixdevs/jobs/blob/master/fullstack/challenge.md)

### Feats:
- Registra tarefas
- Ordena de acordo com data (desc | asc)
- Autenticação via Github

### Ferramentas:
- Next.js (UI)
- MongoDB (Database)
- Next Auth (Auth)
- Vercel (Deploy)
- Typescript
